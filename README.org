A robust implementation of univariate Linear Regression in Julia. This project predicts car prices based on mileage using Gradient Descent. It features a clean, modular architecture with strict separation of concerns between data, logic, and configuration.

* ğŸ“Œ Overview
This project solves the classic linear regression problem: finding the optimal line $\theta_0 + (\theta_1 \times \text{km})$ to minimize the error between predicted and actual car prices.

**Key Features:**
- *Vectorized Implementation:* Efficient mathematical operations using Julia.
- *Environment Management:* Uses ~Project.toml~ for reproducible dependency tracking.
- *Configurable:* Centralized configuration for paths and hyperparameters.
- *Automated Pipeline:* Full support for ~make~ commands to train, predict, and visualize.

* ğŸ§® Theoretical Background

The model minimizes the *Mean Squared Error (MSE)* cost function:

$$J(\theta_0, \theta_1) = \frac{1}{2m} \sum_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)})^2$$

Where the hypothesis $h_\theta(x)$ is given by:

$$h_\theta(x) = \theta_0 + (\theta_1 \cdot x)$$

The parameters are updated using *Gradient Descent*:

$$\theta_j := \theta_j - \alpha \frac{\partial}{\partial \theta_j} J(\theta_0, \theta_1)$$

Where $\alpha$ is the learning rate.

More detailed information on technical aspect of project can be found in [[file:./docs/theory.pdf][theory.pdf]].
